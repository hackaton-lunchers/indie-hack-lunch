# Lunchers
Fancy app for automatic sending favourite restaurants daily menu right into your slack account.

##Content
###Frontend part

Using angular, Typescript, Twitter bootstrap, Less and Gulp. Type `gulp` for installation in /front directory.

###Backend part

Using Hapi, io.js and mongodb in MongoLab. Type `npm install` and `npm start` for installation and starting server.

##User guide

1. Login to application with your cn-group slack account username
2. Add favourite restaurants which daily menu you want to get
3. Daily menus of your favourite restaurant will be sent as a direct message
